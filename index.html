Variable Smart Suspension System

Real-time adaptive motorcycle suspension with sensor feedback & stepper motor control

Designed and implemented a real-time adaptive motorcycle suspension system that autonomously adjusts stiffness based on load and road conditions. The system uses a load cell with HX711 amplifier to measure weight, driving a NEMA 23 stepper motor via L298 driver to modify suspension preload — all coordinated by an ATMEGA328P microcontroller. Structural analysis was validated using ANSYS, while circuit design and simulation were done in Proteus Professional.
